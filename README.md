# international-companies
SPA for Companies and Products - React-Redux and NodeJs-Express


# Instalation
For install the server:
* cd Server
* npm install
* npm start
* It runs on port 4000

For install the Client
* cd Client
* npm install
* npm start
* It runs on port 3000

# Technologies
Client: React-Redux
Server: NodeJs-Express
